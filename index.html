أسئلة متكررة

يتكامل تراك مايل مع Salla وZid وShopify وWooCommerce. كما نوفر RESTful API للتكاملات المخصصة.
نعم. يتضمن تراك مايل تطبيقاً للسائقين مع إدارة الطلبات والملاحة وإثبات التسليم وتحصيل COD.
معظم العملاء يصبحون جاهزين خلال 1-2 أسبوع مع دعم الربط وإعداد المستودعات وتدريب السائقين.
بالتأكيد. نوفر بيئة تجريبية حية. تواصل مع فريق المبيعات لجولة مخصصة.
